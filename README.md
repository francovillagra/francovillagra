## 📌 Proyectos Destacados

**recon-scope** — Plataforma de reconocimiento automatizado para auditorías de seguridad autorizadas. Enumera subdominios, escanea puertos y detecta tecnologías. FastAPI + Next.js.
🔗 https://github.com/francovillagra/recon-scope · Demo: https://recon-scope.vercel.app

**web-vulnerability-scanner** — Scanner de vulnerabilidades web: detecta SSL inválido, security headers faltantes, cookies inseguras, XSS y SQLi.

**auth-api-security** — API REST endurecida con JWT, rate limiting, security headers, bcrypt y validación con Zod.
🔗 https://github.com/francovillagra/auth-api-security · Demo: https://auth-api-production.vercel.app
