## About Me

I'm a Full Stack Developer with a strong focus on security, building web applications and offensive/defensive security tooling. I study Cyberdefense at **FADENA** and work as an external auditor at **AGN (Auditoría General de la Nación Argentina)** — a background that shapes how I approach risk, compliance, and system integrity.

My work lives at the intersection of software engineering and cybersecurity: hardened APIs, reconnaissance platforms, vulnerability scanners, and security monitoring dashboards — all built with production-grade standards.

### 🛰️ Recon Scope
Automated reconnaissance platform for authorized security audits. Enumerates subdomains, scans ports, and fingerprints technologies — with mandatory domain ownership verification before any scan runs.

`FastAPI` · `Python` · `Next.js` · `PostgreSQL` · `Railway`

### 🛡️ Web Vulnerability Scanner
Web vulnerability scanner that detects missing security headers, invalid SSL, insecure cookies, XSS vectors, and SQL injection exposure. Stateless, with a clean results dashboard.

`Next.js` · `TypeScript` · `Zod` · `Recharts`

[![Repo](https://img.shields.io/badge/Code-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/francovillagra/web-vulnerability-scanner)
